Everything in this folder is, and always will be, CRITICAL to the core functions of Cappuccino Framework. These act as the building blocks to many, many features of the framework.

Without CEditor, the Visual Scripting classes for Unity 2019+ (CGraph, CGraphInstance) would no longer work and a majority of the foundations that developers may rely on for quality-of-life would be absent.

Without CObject, CProperty cannot fulfil any Hierarcical Navigation requests and would be partly unreliable.

Without CProperty, any quality-of-life changes that may make the process of getting property information displayed easier can be drastically harder. This is especially the case with additions such as the CProperty Array Extension.
