----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
	[CAPPUCCINO EDITOR FRAMEWORK ->  C# Extensions]
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

	These extensions are macros or otherwise extra methods that use C#'s built-in malleable nature in order to provide some useful extra methods for Cappuccino to leverage
	with it's built in language interpreters. These aren't expected to be used in runtime games applications and don't have any considerations for such, though some care
	has been taken to keep the methods as efficient as possible.
